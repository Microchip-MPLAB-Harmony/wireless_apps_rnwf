3.1.6 Creating a New Project

  1. To create a new project, open MPLAB X IDE and click on File>New Project
  2. In the “New Project” pop-up, click on Microchip Embedded>32-bit MCC Harmony Project>Next
  3. In Framework Selection window, select the Framework path and click Next.
  4. In Project Settings window, enter the folder name and project name and click Next.
  5. In Configuration Settings window,
    • Enter a Name for the configuration setting folder.
    • select the Target device name
    • click Finish.
  6. Now MPLAB X Code Configurator will be launched automatically.
    • Click on Select MPLAB Harmony
    • In the next window, select the latest versions of below mentioned repositories and click Finish. Then the content manager will start downloading.
      • csp
      • core
      • dev_packs
      • bsp
      • wireless_apps_rnwf
      • wireless_system_rnwf
